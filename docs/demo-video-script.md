# Demo Video Script — Condo Manager OS
**Duration:** 2-3 minutes | **Format:** Screen recording with voiceover

---

## Scene 1: Hook (0:00 - 0:15)
**Screen:** Landing page hero section
**Voiceover:**
> "Managing a condominium is a nightmare of spreadsheets, late payments, and angry owners. What if your AI could handle all of it?"

## Scene 2: The Problem (0:15 - 0:30)
**Screen:** Show a messy Excel spreadsheet (use the Coral Breeze file as example, blurred)
**Voiceover:**
> "Most property managers spend 8+ hours a month on accounting alone — calculating shares, chasing payments, writing reminder letters. And one mistake means angry co-owners."

## Scene 3: One Command Setup (0:30 - 0:50)
**Screen:** Terminal running `node setup.js --parent-page=xxx`
**Voiceover:**
> "Condo Manager OS sets up your entire operation in under 5 minutes. One command creates 10 interconnected Notion databases — units, accounting, budget, maintenance, everything. Zero manual setup."

**Screen:** Quick montage of Notion databases appearing

## Scene 4: Dashboard (0:50 - 1:10)
**Screen:** Terminal running `condo-cli.js dashboard`
**Voiceover:**
> "Instant overview — cash position across all accounts, every unit's balance at a glance, delinquencies flagged in red. This is real data from a real condominium."

## Scene 5: Fee Call (1:10 - 1:30)
**Screen:** Terminal running `condo-cli.js fee-call Q1 2026`
**Voiceover:**
> "Quarterly fee calls? One command. It calculates each unit's share based on ownership percentage, creates the accounting entries, and updates all balances automatically."

## Scene 6: Natural Language (1:30 - 1:50)
**Screen:** Telegram chat with OpenClaw assistant
**Chat messages:**
- User: "Who owes money?"
- AI: [Shows delinquency report]
- User: "Send a reminder to A-1"
- AI: [Shows draft reminder letter]

**Voiceover:**
> "Or just talk to it. Ask who owes money, generate a report, record a payment — in Spanish, English, or French."

## Scene 7: Excel Import (1:50 - 2:05)
**Screen:** Terminal running `import-excel.js data.xlsx`
**Voiceover:**
> "Already have data in spreadsheets? The import tool auto-detects your sheets and migrates everything — units, ledger entries, expenses, bank movements."

## Scene 8: CTA (2:05 - 2:30)
**Screen:** Landing page pricing section
**Voiceover:**
> "Condo Manager OS. Built from real property management experience. Available now on ClawHub — free tier to try, Pro for the full suite. Link in the description."

**Screen:** Show URL: casedamare.github.io/condo-os

---

## Production Notes
- **Music:** Lo-fi or ambient electronic, subtle
- **Transitions:** Clean cuts, no fancy effects
- **Terminal:** Use dark theme, increase font size to 16pt
- **Notion:** Show actual Notion UI with the databases
- **Telegram:** Real conversation screenshot (can use Coral Breeze data, anonymize names)
- **Record with:** OBS or Loom
- **Upload to:** YouTube (unlisted or public) + embed on landing page
